# cs608-recsys
Repo for CS608 - Recommender System (Project 2)

## Setup
1. Clone repo

```
git clone https://github.com/davidcjw/cs608-recsys.git
```

2. Change directory

```
cd cs608-recsys
```

3. Create new environment
4. Install requirements

```
pip install -r requirements.txt
```

5. Run streamlit app

```
streamlit run app.py
```
